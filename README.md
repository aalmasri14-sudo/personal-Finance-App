# 💰 Personal Finance App

A comprehensive Android application developed in Java to help users manage their personal finances efficiently. The application provides a complete solution for tracking income and expenses, monitoring financial activities, generating reports, and visualizing financial data through interactive charts.

---

## 📱 Overview

Personal Finance App is designed to simplify money management by allowing users to record daily financial transactions, monitor spending habits, and gain valuable insights into their financial status through detailed reports and graphical analysis.

---

## ✨ Features

### Income Management

* Add income records
* Edit existing income entries
* Delete income transactions
* View income history

### Expense Management

* Add expense records
* Edit expense transactions
* Delete expense records
* Track spending activities

### Search & Filtering

* Search transactions by keyword
* Filter transactions by type
* Filter transactions by date
* Quick access to specific records

### Reports & Analytics

* Financial summary reports
* Income vs Expense comparison
* Transaction statistics
* Monthly financial overview

### Charts & Visualization

* Interactive charts
* Financial trend analysis
* Spending distribution visualization
* Income and expense comparison graphs

### Recent Activities

* Display latest transactions
* Quick review of wallet activity
* Real-time updates
